# house-price-prediction-ml
Machine learning project using Random Forest to predict house prices and property types. Includes end-to-end pipeline and Streamlit app for real-time predictions.
Built machine learning model using Random Forest to predict house prices and property types
Developed end-to-end pipeline including data cleaning, preprocessing, and feature engineering
Performed model evaluation and achieved high prediction accuracy
Designed and deployed an interactive Streamlit web application for real-time predictions
